# StayEase 🏠

**StayEase** is a full-stack property listing web application inspired by Airbnb,
built using the **MERN stack**.  
The platform allows users to explore properties, create listings, and share
reviews, with a focus on clean architecture and scalability.

## 🚀 Features (Implemented)

- 🏡 Property listing with images, price, location, and description
- ➕ Create, update, and delete listings (CRUD)
- ⭐ Review & rating system for properties
- 💬 Commenting system for user feedback
- 🔐 Secure backend APIs with structured data models
- 📦 MongoDB schema design for listings and reviews
- Integrated GeoMaps to enable location-based features, allowing users to view and interact with geographic data seamlessly.
- and many more things yet to come some silent features.

## 🛠️ Tech Stack

- **Frontend:** React, JavaScript, HTML, CSS ,ejs
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **API Style:** RESTful APIs
- **Version Control:** Git & GitHub
